Works with Home Assistant

Built for the
Open Home

Linus is a native Home Assistant device. No third-party integrations, no cloud accounts, no lock-in. It uses the Sendspin protocol — the Open Home Foundation's open standard for multiroom audio.

  • Appears automatically in Home Assistant via ESPHome
  • Stream from Spotify, BBC Sounds, local files and more
  • Control from any HA dashboard, Alexa, or Google Home
  • OTA firmware updates — always improving
  • No subscription, no cloud, fully local
